# Hola, mundo

C++ es un lenguaje **compilado**. Tiene 3 fases:

1. Compilado
2. Vinculación (linking)
3. Ejecución

Un programa básico en C++ se ve así:

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, world!" << std::endl;
    return 0;
}
```
