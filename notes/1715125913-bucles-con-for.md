# Bucles con for

dea

```python
for i in range(5): # 0, 1, 2, 3, 4
	# ...

for i in range(2, 8): # 2, 3, 4, 5, 6, 7
	# ...

for i in range(1, 9, 2): # 1, 3, 5, 7
	# ...

for i in range(10, -5, -3): # 10, 7, 4, 1, -2
	# ...

my_list = ["foo", "bar", "baz"]

for s in my_list: # "foo", "bar", "baz"
	# ...
```
